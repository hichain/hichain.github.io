# [hichain.jp](https://hichain.jp)

Official Website of Hichain Project

## Requirements

- Ruby v3.0.0
- bundler v2.2.16

## Usage

### Installation

1. Install Ruby
2. `$ gem install bundler -v {version}`
3. `$ bundle`

### Update dependencies

1. update Ruby and RubyGems
   1. install latest version of ruby and `$ gem update --system`
2. update bundler
   1. `$ gem update bundler`
3. update gems
   1. `$ bundle update`
